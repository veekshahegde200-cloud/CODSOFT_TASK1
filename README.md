**CodSoft Java Development Internship**

## 📖 Project Description
This project is a console-based **Number Guessing Game** developed in Java as part of the **CodSoft Java Development Internship**.

The program generates a random number between **1 and 100**, and the user must guess the correct number within a limited number of attempts. After each guess, the program provides feedback indicating whether the guess is **too high**, **too low**, or **correct**. The game also supports multiple rounds and keeps track of the user's score.

## ✨ Features
- Generates a random number between 1 and 100.
- Accepts user input for guessing the number.
- Provides hints (Too High / Too Low).
- Limits the number of attempts.
- Supports multiple rounds.
- Displays the user's score.

## 🛠️ Technologies Used
- Java
- Scanner Class
- Random Class

## ▶️ How to Run
1. Clone or download this repository.
2. Open the project in any Java IDE (VS Code, IntelliJ IDEA, Eclipse, or NetBeans).
3. Compile the program:
   ```
   javac NumberGuessingGame.java
   ```
4. Run the program:
   ```
   java NumberGuessingGame
   ```
